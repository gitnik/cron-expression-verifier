How to use

Request 
        POST tremendous.ooo/cron
    
    
        {    
            "cron": "0 0 * * WED"
        }
Response 
        HTTP 200
    
    
        {    
            "nextRun": "2020-06-17T00:00:00.000Z",
            "status": 200
        }
Failing to send a valid cron expression will yield the following response: 
        
            HTTP 400

            {
                "error": "Invalid cron expression. Check crontab.guru for help.",
                "status": 400
            }
